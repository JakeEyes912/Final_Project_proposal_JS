# Title
PONG

## Repository
https://github.com/JakeEyes912/Final_Project_proposal_JS.git

## Description
I will recreate the famous game Pong in VS Code using Python. I find this to be relevant to digital arts and media considering Pong was one of the foundations for games and media in general.
 
## Features
- I will create two moveable characters, one the is controlled and the other that is AI
- I will have a ball that bounces between the controlled and AI characters
- If the ball falls out of the screen, then I will increase the corresponding player's score to whoever's side it landed on

## Challenges
- I will need to understand how to allow player control using VS code
- I will need to figure out how to make an AI move on it's own
- I will need to learn how to make a ball bounce off collisions

## Outcomes
I intend to create a game that will have one player vs an AI pass a ball to each other and implement a score system for whoever gets the ball on the other side.

## Viable Outcome
The bare minimum I need is a player controller and an AI controller that pass a ball between each other
